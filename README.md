# CellOracle
CellOracle is a python library for the analysis of Gene Regulatory Network with single cell data.

For more information, please read our bioarxiv preprint: [CellOracle: Dissecting cell identity via network inference and in silico gene perturbation](https://www.biorxiv.org/content/10.1101/2020.02.17.947416v1).
